
### 一个做记忆宫殿扑克牌训练的辅助app，随机生成52张扑克牌。

记忆宫殿通过记忆扑克牌来训练。虽然扑克牌很小，但是平时随身带上一副扑克牌出来记也不方便。找了找也没有相关的纸牌生成的app，所以自己做了一个demo。
